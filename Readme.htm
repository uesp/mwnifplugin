NifPlugin v0.11 Beta (Build 135)


Introduction

NifPlugin is a 3DSMax v4.0 plugin which allows 3DS to load v3.0 NIF model files used in the ES:Morrowind computer game. This utility is unofficial and not supported by Bethesda or Discreet in any manner.

It is important to note that this initial version is considered Beta and is far from finished. See the features section below for what currently is and isn't supported.

For the latest release along with source code and documentation, see the MWEdit Files area on Source Forge.


Installation

To install the plugin, simply copy the NIFPLUGIN.DLI (merely a Windows DLL file renamed to DLI) into your 3DSMax PLUGINS directory. If 3DSMax is currently running you can start the plugin manually by choosing the Customize->Plug-in Manager... menu. If 3DSMax is not running it will find and load the plugin automatically when it starts.

If you are updating the plugin from a previous version you will have to quit 3DSMax before you will be allowed to overwrite the plugin file.

Once the plugin is installed you can then load NIF files from the File->Import menu.


Features

    Supported Features
  • NIF import plugin for 3DSMax v4.0 (not yet confirmed to work in other versions)
  • Import all mesh data
  • Texture UV coordinates
  • Groups and hierarchies
  • Material properties, including alpha and texture maps

    Unsupported Features
  • Animations (getting there)
  • Root Collision Node
  • Models that include the bitmap data within the NIF file
  • Lights and cameras
  • Vertex coloring
  • Particles
  • Gravity


Known Issues

  • Smoothing groups are not properly implemented. The models should appear smoothed properly but you will notice that there is only one smoothing group assigned to each object. This may cause problems if you wish to modify the smoothing.
